# 中医辨证分析技能

## 任务目标

基于用户的文字与图片线索，从“阴阳—五行—脏腑经络—气血津液—五运六气”的角度进行辨证分析，给出结构化报告；若出现明显病症线索，则给出调理建议与就医提示。

## 输入信息

用户可能提供：

- 主诉/症状：部位、性质（疼/胀/痒/麻等）、程度、持续时间、诱因
- 伴随表现：口苦/口干/口臭、便秘/腹泻、怕冷/怕热、失眠、精神状态等
- 舌象/面色/精神状态（来自图片时尤需说明“仅作参考”）

如果图片无法被有效理解（例如模型未开启视觉或图片质量不足），需明确说明并以文字为主。

## 输出要求（必须包含以下部分）

输出为 Markdown，并按顺序给出：

1. `# 辨证分析（中医视角）`
2. `## 辨证结论`：一句话总结“偏阴/偏阳/偏热/偏寒/偏湿/偏燥/气机郁滞”等可能倾向，并给出主要证候（如：肝郁气滞夹湿热、脾失健运痰湿内阻等）
3. `## 证候要点`：用 3-6 条要点列出证候依据（来自文字线索与图片线索分别说明，若未知则写“信息不足”）
4. `## 可能原因（参考）`：从起居饮食、情志、外感内伤、体质与季节气候变化角度简述
5. `## 调理建议`：给出可执行建议，分为
   - 日常调理（起居饮食、作息、运动）
   - 情志调护（如：疏肝理气/宁心安神/健脾养胃等方向性建议）
   - 观察要点（2-4 条：如舌苔变化、大便性状、睡眠情况等）
6. `## 注意事项`：必须包含安全边界（不能替代医生；急症需就医；不建议自行用药/中药方剂）

## 辨证思路（写作约束）

- 先做“阴阳倾向”，再做“寒热湿燥”判断；再定位到脏腑与气机环节（如：肝主疏泄/脾主运化/心主神明/肾藏精）。
- 五行仅做“关联性解释”，避免绝对化结论。
- 五运六气只用于“季节气候背景的可能性提示”，不要强行断言。

## 风险控制

- 本技能输出为健康管理与参考，不提供确诊与处方。
- 遇到以下情况必须提示就医：高热、呼吸困难、持续剧烈疼痛、呕血便血、意识改变、孕产人群严重症状等。

## 示例（简化）

### 示例 1（偏热/肝火或胃火线索）

输入：口苦、口干、便秘、烦躁、睡不着。

输出：

# 辨证分析（中医视角）
## 辨证结论
偏热、夹肝郁与胃腑火炽的倾向。
...

### 示例 2（偏寒/脾阳不足线索）

输入：怕冷、腹泻、食少、舌淡。

输出：

## 辨证结论
偏寒、脾阳不足的倾向。
